CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web.

Tipos de Seletores CSS
Elemento Utilizou? Explicação
Universal check_circle_outline Aplica o estilo a todos os elementos
Tipo check_circle_outline Aplica o estilo aos marcadores de certo tipo
Classe check_circle_outline Aplica o estilo a uma classe
Identificador check_circle_outline Aplica o estilo a um item específico
Atributos close Aplica o estilo consoante o atributo de um elemento
Pseudo-classe close Aplica o estilo consoante o estado de um elemento
Pseudo-elemento close Aplica o estilo consoante as partes de elementos

Combinadores de Seletores CSS
Elemento Utilizou? Explicação
Descendentes check_circle_outline div p – todos os p que estão “dentro” de um div
Filhos check_circle_outline div > p – todos os p que estão “logo dentro” de div
Adjacentes check_circle_outline div ~ p – todos os p que estão colocados depois de um div e foram usados nas páginas principais
Imediatamente adjacentes check_circle_outline div + p – todos os p que estão colocados depois de um div
Agrupação check_circle_outline div,p{ color: blue; } - os div e p ficarão com fonte azul
